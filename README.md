# 📊 Data Analytics Project

## Overview
Este projeto tem como objetivo analisar dados de comportamento de clientes para identificar padrões, tendências e insights que possam apoiar decisões de negócio. O projeto abrange todo o fluxo de trabalho de análise de dados, incluindo preparação dos dados, análise exploratória, consultas em banco de dados e visualização de resultados por meio de um dashboard interativo.

O objetivo principal é transformar dados brutos em **insights acionáveis**, utilizando ferramentas amplamente utilizadas na área de **Data Analytics**.

---

## Dataset
O conjunto de dados contém informações sobre compras de clientes, incluindo características demográficas, detalhes das compras e comportamento de consumo.

**Principais tipos de informações:**
- Dados demográficos dos clientes (idade, gênero, localização)
- Informações sobre compras (produto, categoria, valor da compra)
- Comportamento de compra (frequência, descontos, avaliações)
- Informações de envio e métodos de pagamento

Esses dados são utilizados para identificar padrões de consumo, preferências de produtos e segmentação de clientes.

---

## Tools
As seguintes ferramentas e tecnologias foram utilizadas no projeto:

- **Python** – preparação, limpeza e análise exploratória de dados  
- **Pandas & NumPy** – manipulação e transformação de dados  
- **SQL (PostgreSQL / MySQL / SQL Server)** – análise de dados e consultas em banco  
- **Power BI** – criação de dashboard interativo  
- **Gamma / PowerPoint** – criação da apresentação do projeto  
- **GitHub** – versionamento e organização do projeto  

---

## Project Steps

### 1. Data Loading
O dataset foi carregado utilizando Python e a biblioteca **Pandas** para iniciar o processo de análise.

### 2. Data Cleaning
Foram realizadas etapas de limpeza de dados, incluindo:
- tratamento de valores ausentes  
- remoção de duplicidades  
- padronização de colunas  
- verificação de consistência dos dados  

### 3. Exploratory Data Analysis (EDA)
A análise exploratória foi realizada para compreender melhor os dados e identificar padrões importantes, como:
- distribuição de compras
- comportamento de clientes
- produtos mais vendidos
- relação entre descontos e compras

### 4. SQL Analysis
Os dados foram carregados em um banco de dados para realizar consultas SQL e responder a perguntas de negócio, como:
- clientes que mais geram receita
- categorias de produtos mais vendidas
- impacto de descontos nas vendas
- segmentação de clientes

### 5. Data Visualization
Os resultados da análise foram transformados em visualizações interativas utilizando **Power BI**, permitindo explorar os dados de forma clara e intuitiva.

### 6. Reporting
Os principais insights foram documentados em um **relatório analítico**, destacando descobertas relevantes e recomendações de negócio.

### 7. Presentation
Foi criada uma **apresentação em PPT utilizando Gamma**, resumindo os principais resultados e insights do projeto para comunicação com stakeholders.

---

## Dashboard
O dashboard desenvolvido no **Power BI** apresenta visualizações interativas que ajudam a explorar os dados de forma dinâmica.

Principais elementos do dashboard:

- Total de vendas
- Tendência de compras ao longo do tempo
- Produtos mais vendidos
- Segmentação de clientes
- Impacto de descontos nas vendas
- Análise por categoria de produto

O dashboard permite que usuários filtrem informações e explorem diferentes perspectivas dos dados.

---

## Results
A análise permitiu identificar insights importantes, como:

- padrões de compra dos clientes  
- categorias de produtos com maior desempenho  
- impacto de descontos no volume de vendas  
- segmentação de clientes com maior valor para o negócio  

Esses resultados podem ajudar a empresa a melhorar estratégias de marketing, otimizar campanhas promocionais e tomar decisões baseadas em dados.
